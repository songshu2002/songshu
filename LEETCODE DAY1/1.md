1.连环遍历i a
且去除两次数组取到相同序号的情况
2.制定新数组来输出结果



+++

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
  int[] result = new int[2];
  for(int i = 0;i<nums.length;i++){
      for(int a = 0;a<nums.length;a++){
          if(i!=a && nums[i]+nums[a]==target){

          
          result[0]=a;
          result[1]=i;

      }
  }
    }
    return result;
}
}
```
