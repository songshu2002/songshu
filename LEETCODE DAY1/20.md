push() 方法可向数组的末尾添加一个或多个元素，并返回新的长度。

charAt() 方法可返回指定位置的字符。

pop() 方法用于删除数组的最后一个元素并返回删除的元素。

peek()返回第一个元素

***



1.创建字符栈

2.遍历输入时的字符串的长度

3.如果出现以下三种字符（左边框），就压入栈内

然后分别对3种符号的（右边框）展开讨论：

如果栈内没有这个符号且栈内首元素与此符号左边框

不同，就出栈，否则返回嵌套false。

```java
class Solution {
    public boolean isValid(String s) {
   Stack<Character> stack = new Stack<>();
        for(int i=0;i<s.length();i++){
            char c = s.charAt(i);
            switch(c){
                case '(':
                case '{':
                case '[':
                    stack.push(c);
                    break;
                case ')':
                    if(!stack.isEmpty()){
                        if('('==stack.peek()){
                            stack.pop();
                            break;
                        }else{
                            return false;
                        }
                    }else{
                        return false;
                    }
                case ']':
                    if(!stack.isEmpty()){
                        if('['==stack.peek()){
                            stack.pop();
                            break;
                        }else{
                            return false;
                        }
                    }else{
                        return false;
                    }
                case '}':
                    if(!stack.isEmpty()){
                        if('{'==stack.peek()){
                            stack.pop();
                            break;
                        }else{
                            return false;
                        }
                    }else{
                        return false;
                    }
            }
        }
        return stack.isEmpty();

}     
}
```
